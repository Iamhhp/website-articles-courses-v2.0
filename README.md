## Installation:

### Clone the project:

`git clone` https://github.com/Iamhhp/website-articles-courses-v2.0.git

### Install dependencies:

`npm install`

### Backend with liara.ir:

[backEnd] (https://dbserver.liara.run/)

### Start React App:

`npm start`
